# ZAYRO - レストランレビュー管理システム

ZAYROは、レストランのオーナーがGoogle、Yelp、TripAdvisorなど複数のプラットフォームのレビューを一元管理できるダッシュボードシステムです。

## 主な機能

- 複数プラットフォームのレビュー統合表示
- 評価分析と統計情報
- レビュー返信管理
- 多言語対応
